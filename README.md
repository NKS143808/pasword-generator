# 🔐 Password Generator

A simple Python application that generates strong random passwords.

## Features

- Generate secure passwords
- Custom password length
- Includes:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters

## Technologies Used

- Python

## How to Run

```bash
py main.py
```

## Example

```
===== PASSWORD GENERATOR =====
Enter password length: 12

Generated Password:
A#9xP!2mQ@7L
```
